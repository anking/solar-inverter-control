Control
Output UTI — Grid First
Grid powers loads. Battery is protected from discharge. Solar still charges the battery. SBU — Solar/Battery/Utility
Solar and battery power loads first. Grid is used only when battery is depleted. SUB — Solar/Utility/Battery
Solar and grid power loads first. Battery is used only when both are insufficient. SOL — Solar Only
Only solar powers loads. No grid or battery usage. Output is off when solar is insufficient.
Charge SNU — Solar & Grid Charging
Battery charges from both solar panels and the grid. Fastest charging, uses grid power. OSO — Solar Only Charging
Battery charges from solar panels only. No grid power used for charging. Maximizes solar self-consumption.
