Auto Mode Configuration

Configure how the inverter automatically manages battery charge levels. Select a profile and adjust the SOC thresholds using the sliders below.

Profile
SMART MODE Automatically switches profiles based on weather forecast
Decision rules (checked every 15 min):
• Thunderstorm in forecast → Pure Backup (charge to max)
• Heavy rain/snow + high precip probability → Pure Backup
• Overcast + rain likely → Balanced (conserve battery)
• Sunny skies + 3h+ sun left + SOC above 60% → Max Solar (drain to harvest)
• Otherwise → Balanced
